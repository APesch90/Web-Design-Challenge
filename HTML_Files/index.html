Summary: Latitude vs. X

The purpose of this project was to analyze data from OpenWeatherMap.org, specifically a dataset of 546 cities, to understand if there were any trends between maximum temperature, humidity, cloudiness, and wind speed as compared to the latitude of the city. Data was captured around the same timeframe, and a .csv was pulled using Open Weather Map's API. Subsequently, plots were made using Matplotlib to help show correlations and further explain observations. Click on each plot for more details on the analysis!
